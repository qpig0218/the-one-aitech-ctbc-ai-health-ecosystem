# 智慧雨林健康照護｜116 年預備計畫骨架 v0.1

定位：本文件作為 116 年度智慧雨林健康照護或相關大南方新矽谷整合型補助的預備骨架。115 年度健康照護領域多已截止，本案應從 2026 下半年開始佈局下一輪。

## 1. 計畫名稱草案

南科 AI 健康促進與企業健康數據治理示範計畫

## 2. 三方協作模式

| 角色 | 單位 | 分工 |
|---|---|---|
| 場域出題 | 南科企業 / ASML / 科技製造業 | 員工健康促進需求、試辦場域、導入驗證 |
| AI 技術導入 | 本一科技 | AI 健康平台、資料治理、dashboard、LineBot/App |
| 人才共育 | 中信科技大學 | 課程、認證、師資、學生實作、健康專業 |

## 3. 計畫背景

1. 大南方新矽谷推動方案強調「擴算力、鏈場域、引人才、展應用」。
2. 南科半導體與高科技產業聚落具有高度員工健康促進需求。
3. 企業需要兼具員工關懷、職安、ESG、留才與數據治理的健康促進解方。
4. 中信科大具備運動健康、心理/輔導、照護與資訊相關資源。
5. 本一科技具備 AI 健康照護、HIS Copilot、TaiOne Care GPT、BodyCam、LineBot 與 dashboard 技術基礎。

## 4. 計畫目標

1. 建立南科企業 AI 健康促進示範服務。
2. 完成 AI 體適能檢測、個人化健康處方與企業 dashboard。
3. 建立中信科大 AI 健康促進人才培育與認證制度。
4. 完成至少 1 家南科企業示範，並擴散至 3-5 家企業。
5. 建立可複製之「企業出題、AI 導入、人才共育」模式。

## 5. 工作項目

| WP | 工作項目 | 主責 |
|---|---|---|
| WP1 | 場域需求盤點與企業健康促進模型 | 本一 + 南科企業 + 中信 |
| WP2 | AI 健康促進平台開發與導入 | 本一 |
| WP3 | 體適能、心理健康、運動處方服務設計 | 中信科大 |
| WP4 | 企業 dashboard 與 ESG/職安報告 | 本一 |
| WP5 | 人才培育課程與認證 | 中信科大 + 本一 |
| WP6 | 示範案驗證與擴散 | 三方 |
| WP7 | 資安、個資與資料治理 | 本一 + 企業 IT |
