# B3 ｜ 台南據點評估備忘錄

**版本**：v1.0｜2026-05-05
**製作**：本一科技
**對象**：5/11 內部決策

---

## 1. 為何要在台南設據點

| 動因 | 說明 |
|------|------|
| **客戶就在台南** | 中信科大、ASML、南科廠商、台南市政府都在台南 |
| **補助要求在地** | 地方型 SBIR 規定設籍台南、執行場所在台南；大南方計畫優先在地 |
| **生態系信號** | 「台南有辦公室」是叩門南科局／市府／辦公室的基本條件 |
| **政府主動推** | 市政府已多次表達希望推進地方型 SBIR，「治理」型 SBIR 過件率高 |
| **校友／關係綿密** | ADER 在地醫療人脈、奇美醫院關係、深耕臺南系列已有累積 |

## 2. 三選項比較

| 項目 | A. TTA 沙崙辦公室 | B. 台南設子公司 | C. 台南設平行公司 |
|------|---------------|---------------|---------------|
| **設立速度** | ✅ 即可使用 | ❌ 4–8 週 | ❌ 8–12 週 |
| **法人關係** | 本一科技台北本部延伸 | 母子（會計合併）| 各自獨立 |
| **設立成本** | NT$ 0（既有） | NT$ 5–15 萬 | NT$ 10–25 萬 |
| **每月固定成本** | 既有 | 子公司會計、額外人事 +5–10 萬/月 | 獨立會計 +8–15 萬/月 |
| **補助申請資格** | ✅ 設籍若可在台南則 OK；TTA 屬高鐵特區（行政區屬歸仁）| ✅ 直接設籍台南 | ✅ 直接設籍台南 |
| **承接地方型 SBIR** | ⚠️ 需確認 TTA 設籍細節 | ✅ 完全符合 | ✅ 完全符合 |
| **承接國科會南科業務** | ✅ | ✅ | ✅ |
| **稅務規劃彈性** | 一致 | 中（盈餘可移轉）| 高（完全獨立）|
| **品牌一致性** | 高 | 中 | 低（要做雙品牌） |
| **後續退場** | 無痛 | 子公司可關 | 麻煩 |

## 3. 推薦路徑：**「TTA 先用 → 子公司分階段建立」**

### Phase 1（立即，5–7 月）：TTA 沙崙當行政地址
- 既有資源、零成本
- 對外就稱「本一科技 ｜ 台南據點 ｜ TTA 沙崙」
- 用於：對中信科大會議、ASML 拜訪、對市府／南科辦公室名片打點

### Phase 2（7–9 月，視 ASML 案進度）：評估子公司
- **觸發條件**：地方型 SBIR 過件 OR ASML 簽 MOU OR 累積本季營收 NT$ 200 萬+
- **設立決策點**：8 月底
- **建議形式**：B 子公司（盈餘可回母公司、會計合併、較有彈性）

### Phase 3（10 月起）：正式啟用台南子公司
- 設籍南科辦公室周邊或安南商圈（看人才聚集）
- 編制：台南端 1–3 人駐點（業務 + 客戶成功 + PM）

## 5. 風險與緩衝

- **風險 A**：TTA 設籍不符地方型 SBIR → **緩衝**：先用台南合作律師事務所掛址，子公司晚一步成立
- **風險 B**：子公司設了之後業務量不夠 → **緩衝**：Phase 2 設定明確 Go/No-Go 觸發條件，不貿然設立
- **風險 C**：人才不易找 → **緩衝**：先用台北端遠端 + TTA 駐點輪派，第二步再實體編制
