## 一、生態系核心定位

> **「南台灣 AI 健康促進產官學研聯盟」**
> 由 The One AITech（本一科技） × 中信科大為發動軸心，以南科廠商為應用場域，以南台灣為示範區。

```
       媒體 / 公關
            │
            ▼
產官 ─→ The One AITech×中信 ─→ 學研
            │
            ▼
      南科廠商生態
            │
            ▼
   社區 / 樂齡 / 員工家庭
```

## 七、生態系飛輪的關鍵風險

| 風險 | 緩衝 |
|------|------|
| 第一個示範案（ASML）若延遲 → 整個飛輪停轉 | 同步啟動「全民運動中心 / 中信校內」第二場域，作為 Plan B 示範 |

| 廠商鑑定服務被視為「賣認證」 | 強調「中信學術背書 + 真實 ROI 量化」差異化 |
| 中信內部跨單位整合慢 | 教務長已啟動跨單位推動小組（PDF 第 13 頁） |
| 媒體公關過早 → 案子還沒做出來反被打臉 | 5–8 月暫不主動，9 月後才公關 |
